======================================================================
  TROUBLESHOOTING
======================================================================

If hamburger menu doesn't appear on mobile:
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh (Ctrl+F5)
3. Check browser width is <768px
4. Verify mobile-menu.js is loaded

If desktop header looks wrong:
1. Check browser width is >1024px
2. Clear browser cache
3. Verify mobile-optimized.css loaded correctly
4. Check for console errors (F12)

If layouts don't respond:
1. Check viewport meta tag in <head>
2. Verify CSS file paths are correct
3. Clear cache and reload
4. Test in incognito mode
